# Solis Hybrid Inverter - ESPHome

ESPHome конфігурація для моніторингу та керування гібридним інвертором Solis RHI-5K-48ES-5G через Modbus RTU.

## Обладнання

- ESP32-S3 DevKit
- MAX485 модуль (RS485)
- Solis RHI-5K-48ES-5G інвертор

## Підключення

- TX: GPIO17
- RX: GPIO16
- Modbus: 9600 baud, адреса 1

## Веб-інтерфейс

Після прошивки ESP32, веб-інтерфейс буде доступний за адресою:
- `http://solis-hybrid-full.local/` (якщо підтримується mDNS)
- або за IP адресою ESP32 у вашій мережі

**Логін:** admin
**Пароль:** admin

⚠️ **Важливо:** Змініть пароль у файлі `solis.yaml` в секції `web_server` перед використанням!

## Конфігурація WiFi

Перед прошивкою відредагуйте `solis.yaml`:
```yaml
wifi:
  ssid: "YOUR_WIFI_SSID"
  password: "YOUR_WIFI_PASSWORD"
```

Якщо ESP32 не зможе підключитися до WiFi, він створить точку доступу:
- SSID: `Solis-Hybrid-Fallback`
- Пароль: `12345678`

## Функції

### Моніторинг
- PV панелі (напруга, струм, потужність)
- Батарея (напруга, струм, SOC, SOH, температура)
- Мережа (напруга, струм, частота, потужність)
- Backup/Навантаження
- Лічильники енергії
- Температура інвертора
- Статус системи

### Керування
- SOC мінімальний/максимальний
- Грід-ліміт експорту
- Ліміт струму заряду від мережі
- ESS режими (Self Use, Feed-in First, Backup, Manual)
- Дозвіл заряду від мережі

## Прошивка

```bash
esphome run solis.yaml
```

Або використовуйте ESPHome Dashboard для прошивки через веб-інтерфейс.
